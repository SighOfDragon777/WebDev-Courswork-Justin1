# Unit 10 Reflection

In this unit, I continued practicing how to apply HTML, CSS, and JavaScript together in one project. This helped me see how the three languages work as a complete system for building web pages. I also learned how to implement a navigation bar. This included understanding what a navigation bar is, how to create it using HTML, how to style it with CSS, and how to link it to different pages. Navigation is essential for any multi-page website, and I feel more confident building it now.The third lesson introduced design considerations and best practices. I learned about principles such as visual hierarchy, consistency in design, navigability, and color theory with typography. These concepts help make a website not only functional but also user-friendly and professional.Overall, this unit showed me that building a website is not just about making things work — it also requires thoughtful design and easy navigation. I still need more practice combining all three technologies smoothly, but I now have a clearer framework to follow.
